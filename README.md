# StoryQuest

AI 绘本闯关：上传单词图片，自动生成绘本故事与闯关卡，孩子在故事中学英语。

## 技术栈

Next.js、Postgres（Drizzle）、Better Auth、Stripe、i18n、Tailwind 等。

## 开发

```bash
pnpm install
pnpm dev
```

## 部署

- 主分支适用于 Vercel 或 Docker，默认使用 Postgres。
- 其他部署方式请参考项目文档。
